How to use the churn-service

This is a example about how to do a request in with python

        
import requests
url = 'https://churn-service-jhcs.herokuapp.com/predict'
customer = {"contract": "two_year", "tenure": 12, "monthlycharges": 10}
prediction = requests.post(url, json=customer).json()
print(prediction)

You can run by script or in jupyter notebook the code above

Try yourself

Contract: Indicates the customer’s current contract type: Month-to-Month, One Year, Two Year.

Tenure in Months: Indicates the total amount of months that the customer has been with the company.

Monthly Charge: Indicates the customer’s current total monthly charge for all their services from the company.
